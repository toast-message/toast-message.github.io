<toast-message>

Loading https://toast-message.github.io/element.js will inject a <toast-message> Tag at the end of the document (no need to add it yourself)

The Web Components listens at document level for a toast-message Event

The Event details Object takes a message [String] and optional options [Object]

example:document.dispatchEvent(new CustomEvent("message", { delay:1000} ))

There is also a matching global function showToastMessage( String, Object ) for ease of use
